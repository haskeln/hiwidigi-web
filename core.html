API Quickstart

Use one endpoint to resolve an intent deterministically by context and policy.

  -H "Content-Type: application/json" \
  -d '{ "scenarioId": "provider-stripe-firestore" }'

# Example response
{
  "selection": {
    "payment": "stripe_payment",
    "logistics": "fedex_logistics",
    "datastore": "firestore_datastore"
